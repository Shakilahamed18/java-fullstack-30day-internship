# Day 04 – JavaScript Core

## Learning Objectives

- Understand JavaScript data types and type coercion
- Write functions using different declaration styles
- Explain scope, hoisting, and closures
- Manipulate arrays and objects effectively
- Use common array methods for data transformation

## Topics Covered

- Data types: string, number, boolean, null, undefined, object, symbol
- Type coercion and `===` vs `==`
- Variables: `var`, `let`, `const` and their scoping rules
- Functions: declarations, expressions, and IIFE
- Scope: global, function, block scope
- Hoisting behavior
- Closures and practical use cases
- Objects: creation, access, methods, `this` keyword
- Arrays: creation, indexing, common methods (`map`, `filter`, `reduce`, `find`, `forEach`)

## Hands-On Task

Build a small JavaScript program (run with Node.js) that manages a list of students:
1. Create an array of student objects with: name, age, grade, and subjects
2. Write functions to:
   - Add a new student
   - Find students by grade (using `filter`)
   - Calculate average age (using `reduce`)
   - Get a list of all unique subjects (using `map` + `Set`)
   - Sort students by name alphabetically
3. Demonstrate closure by creating a counter function that tracks how many students were added

## Practice / Homework

Solve 5 array manipulation challenges using only `map`, `filter`, and `reduce` — no `for` loops allowed. Examples: flatten a 2D array, group items by category, find the most frequent element.
