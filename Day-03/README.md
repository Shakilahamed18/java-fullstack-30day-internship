# Day 03 – CSS

## Learning Objectives

- Understand the CSS box model and how it affects layout
- Build layouts using Flexbox and CSS Grid
- Create responsive designs with media queries
- Apply modern CSS techniques for spacing, typography, and color
- Structure CSS for maintainability

## Topics Covered

- Box model: margin, border, padding, content, `box-sizing`
- Display properties: block, inline, inline-block, none
- Flexbox: `display: flex`, main/cross axis, `justify-content`, `align-items`, `flex-wrap`
- CSS Grid: `grid-template-columns/rows`, `gap`, `grid-area`, `fr` units
- Positioning: static, relative, absolute, fixed, sticky
- Responsive design: media queries, mobile-first approach
- CSS units: px, em, rem, %, vw/vh
- CSS variables (custom properties)

## Hands-On Task

Style the HTML page from Day 02 using CSS:
1. Apply a consistent color scheme using CSS variables
2. Create the navigation bar using Flexbox (horizontal on desktop, hamburger on mobile)
3. Build the services section as a responsive Grid (3 columns on desktop, 1 on mobile)
4. Style the form with proper spacing, focus states, and a styled submit button
5. Add a media query breakpoint at 768px for tablet/mobile layout

## Practice / Homework

Recreate a simple landing page layout from a design reference (e.g., a Dribbble shot) using only Flexbox and Grid — no frameworks. Focus on getting the layout right rather than pixel-perfection.
